# Manifiesto
Actividad Ronald 02/03/2026

Jerarquía del proyecto:

La portada esta acompañada de una frase corta y también con una imágen de fondo para acompañar el texto

Los títulos principales son grandes y con una mayor grosor para captar la atención primero.

Los subtítulos son más pequeños y livianos que los títulos, pero aún se destacan.

Los parrafos y cuerpo de texto son de tamaño mediano y de peso ligero para lectura cómoda.

Los elemnetos visuales (imágenes, videos y botones) están colocados estratégicamente para apoyar el flujo de lectura y mantener el interés.

Por ultimo, los datos del autor acompañado por el nombre más una breve descripción, así como tambien un botón que despliega las redes sociales del propio autor (Instragram, GitHub y Gmail).
